Experimental Camera

Consent Camera. Faces stay pixelated by default, and capture stays locked until each detected face actively consents.

Overview

This camera is built around a refusal. Typical cameras assume access the moment someone enters frame. This project flips that default and makes visibility something that has to be granted. The feed starts pixelated, the interface announces consent status, and capture stays disabled until consent is confirmed.

How it works

A face model tracks faces over time so consent can stick while someone moves. A hand model detects an OK gesture held for a few seconds and grants consent to the nearest face. If not everyone is consented, the interface blocks capture and blocks filters. A small menu stays visible for reset and blur controls, plus an intro popup explains how the camera works before interaction starts.

Reflection

The early prototype was just full-frame pixelation, but it didn’t make the permission logic legible. The final version adds a status banner, warning states, and a live countdown so consent feels deliberate instead of random. A future improvement would be stronger identity tracking and a gesture that can revoke consent, not only grant it.
